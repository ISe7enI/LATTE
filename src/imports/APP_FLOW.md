# 拿铁 (LATTE) - 应用流程文档

**版本**: v1.0  
**最后更新**: 2026年3月15日

---

## 1. 应用整体架构

### 1.1 核心页面结构

```
┌─────────────────────────────────────────────────────────────┐
│                        拿铁 APP                              │
├─────────────────────────────────────────────────────────────┤
│  Tab栏（底部导航）                                           │
│  ┌─────────┬─────────┬─────────┬─────────┬─────────┐       │
│  │  首页   │  计划   │  记录   │  历史   │  我的   │       │
│  │  Home   │  Plan   │  Log    │ History │ Profile │       │
│  └─────────┴─────────┴─────────┴─────────┴─────────┘       │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 页面层级

```
Level 0: 启动页 / 引导页 / 登录页
    ↓
Level 1: 主页面（Tab导航）
    ├── 首页（Dashboard）
    ├── 计划（Training Plans）
    ├── 记录（Training Logger）- 核心
    ├── 历史（History Calendar）
    └── 我的（User Profile）
        ↓
Level 2: 二级页面
    ├── 动作库（Exercise Library）
    ├── 数据分析（Analytics）
    ├── 训练详情（Workout Detail）
    ├── 计划详情（Plan Detail）
    ├── 计划创建（Plan Creator）
    ├── 设置（Settings）
    └── 动作详情（Exercise Detail）
        ↓
Level 3: 三级页面
    ├── 动作3D演示（3D Demo）
    ├── 数据图表（Charts）
    ├── PR记录（PR Records）
    └── 身体数据（Body Metrics）
```

## 2. 核心流程详解

### 2.1 首次启动流程

```
┌─────────────┐
│   启动APP   │
└──────┬──────┘
       ↓
┌─────────────┐     ┌─────────────┐
│  检查是否   │────→│   引导页    │
│  首次使用   │     │  Onboarding │
└──────┬──────┘     └─────────────┘
       │ 否                ↓
       ↓           ┌─────────────┐
┌─────────────┐    │  用户档案   │
│  免责声明   │←───│   设置      │
│  勾选同意   │    └─────────────┘
└──────┬──────┘           ↓
       ↓           ┌─────────────┐
┌─────────────┐    │  隐私政策   │
│  隐私政策   │←───│   勾选同意  │
│  勾选同意   │    └─────────────┘
└──────┬──────┘           ↓
       ↓                  ↓
┌─────────────┐    ┌─────────────┐
│  进入首页   │←───│   完成      │
└─────────────┘    └─────────────┘
```

**Onboarding 内容**:
1. 产品Slogan展示: "Progress is not linear"
2. 核心功能介绍（3-4页）
3. 用户档案设置（身高、体重、年龄、性别、健身目标、训练水平）
4. 单位设置（公制/英制）

- 播放静音音频维持后台运行

- 用户接打电话时计时器继续运行
